问题 3/12

如何修改账户密码？

标准答案

进入"个人设置"页面，点击"安全设置"选项卡，选择"修改密码"，输入当前密码和新密码后提交即可。

RAG系统回答

要修改您的账户密码，请按照以下步骤操作：

  1. 登录到您的账户
  2. 点击右上角的头像或用户名，打开下拉菜单
  3. 选择"个人设置"选项
  4. 在左侧导航栏中，点击"安全设置"
  5. 找到"修改密码"选项并点击
  6. 在表单中输入您的当前密码
  7. 输入新密码并确认
  8. 点击"保存"或"提交"按钮完成修改

修改成功后，系统会自动退出，您需要使用新密码重新登录。
